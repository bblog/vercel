面向对象的概念

类：类是一个模板，它描述一类对象的行为和状态。

类里面定义了方法和属性。方法 == 函数

对象：对象是类的一个实例，有方法和属性。

属性 == 变量== 成员

如下图所示：Dog是一个类，它描述了整个集合的属性和方法，而Dog1、Dog2等是Dog的一个实例。

简而言之，类就是描述了一个包含方法和属性的模板，而对象就是引用某个模板（类）具体例子。

面向对象的三大特征

封装性

在面向对象程式设计方法中，封装（英语：Encapsulation）是指一种将抽象性函式接口的实现细节部分包装、隐藏起来的方法。

封装可以被认为是一个保护屏障，防止该类的代码和数据被外部类定义的代码随机访问。

要访问该类的代码和数据，必须通过严格的接口控制。

封装最主要的功能在于我们能修改自己的实现代码，而不用修改那些调用我们代码的程序片段。

封装的目的就是将一个类的代码与属性隐藏起来，利用该类的方法来改变数据。

封装性的实现方法

使用private关键字来封装，利用public关键字的方法来访问。

private是私有类型本类才能访问，其他类都访问不了，public公有类型，外部可以访问。

属性封装：private 属性类型 属性名称 ;

方法封装：private 方法返回值 方法名称(参数列表);

继承性

继承就是子类继承父类的特征和行为，使得子类对象（实例）具有父类的实例域和方法，或子类从父类继承方法，使得子类具有父类相同的行为。

被继承对象称为父类，继承产生的类是子类或派生类。

子类完全可以使用父类已定义的方法和属性（变量）。

String类

在 Java 中字符串属于对象，Java 提供了String 类来创建和操作字符串。

创建字符串（实例化String类）：

直接赋值

String str = "hello";

第二种：

String str=new String("hello");

还可以这样定义：

char[] hello = { 'h', 'e', 'l', 'l', 'o'};

String hello = new String(hello);

String 类本身是不可改变的，一旦创建了 String 对象，那它的值就无法改变了。改变值只是让他指向了另一个对象。

一个字符串是一个String类的匿名对象，匿名对象是已经在堆中开辟看内存空间并可以直接使用的对象。

String类的方法

字符串相等

注意：

直接赋值String str1=new String("hello");和String str2=new String("hello");两者并不相等

即str1 == str2的结果为false，因为new关键字开辟了新的空间。

解决方法：

不使用逻辑判断符==，使用String类提供的方法： public boolean equals(String Object)

对象与字符串相等返回true，不相等则返回false。

在Java中主要存在4块内存空间，这些内存空间的名称及作用如下。 (1)栈内存空间:保存所有的对象名称(更准确地说是保存了引用的堆内存空间的地址)。 (2)堆内存空间:保存每个对象的具体属性内容。 (3)全局数据区:保存static 类型的属性。 (4)全局代码区:保存所有的方法定义。

上面的代码：p1.country ="B城";中static声明的属性是由一个对象修改的，但是static声明的属性是类所有，也称为类属性，所以static声明的属性最好要用类名来调用，即Person.country ="B城";。

使用static声明方法

static也可以用来声明方法，称为类方法，可以由类直接调用。

比如：Person.setCountry("B城");

非static声明的方法可以调用static声明的方法与属性，但static声明的方法和属性不能调用非static类型声明的方法和属性。
